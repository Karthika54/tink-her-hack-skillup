# **App Name**: SkillLink

## Core Features:

- Home Dashboard: Display user's SkillCoin balance, AI-powered skill suggestions based on learning interests, and a list of nearby mentors.
- Skill Matching Engine: Recommend potential mentors and evaluate potential skill match, and provide smart recommendations and notifications using the AI tool.
- Skill Exchange Modal: Enable users to confirm skill exchanges, deducting coins from the user and adding them to the mentor. Also include animation upon successful exchange.
- Chat Interface: Open a chat screen after skill exchange confirmation with a preloaded message. Provide a basic messaging interface.
- Leaderboard: Display a gamified leaderboard ranking users based on coins earned.
- Session History: List past skill exchange sessions with status, coins, and timestamps.
- User Profiles and Mentor Verification: Hardcode user and mentor data; automatically assign 'Verified Mentor' badge based on the number of sessions taught, and persist information in the Firestore database.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to represent creativity and intellect.
- Background color: Light purple (#F3E5F5) to provide a soft, engaging backdrop.
- Accent color: Blue (#5677FC) for interactive elements and highlights.
- Body and headline font: 'Poppins' sans-serif for a contemporary, precise feel.
- Use consistent, student-friendly icons for skills and interactive elements. Include a sparkle icon to denote AI-suggested content.
- Employ a clean and modern layout using rounded cards and soft gradients to create an inviting user interface.
- Incorporate subtle animations for button clicks and coin updates to enhance the interactive feel.
